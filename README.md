# 🔐 Intelligent Password Evaluator

## 📌 Introduction

This project is a simple AI-inspired system that analyzes the strength of a password based on multiple characteristics. It uses logical rules to simulate how intelligent systems evaluate security levels.

---

## 🎯 Objective

The goal of this project is to determine whether a password is **Weak, Moderate, or Strong** by examining its structure and complexity.

---

## ⚙️ How the System Works

The program evaluates a password using the following criteria:

* Length of the password
* Presence of uppercase letters
* Presence of lowercase letters
* Inclusion of numeric digits
* Use of special characters

Each condition contributes to an overall strength score, which is then used to classify the password.

---

## 🧠 Concept Behind the Project

This project is based on **rule-based decision making**, where predefined conditions are used to simulate intelligent behavior.

Instead of using machine learning, the system applies logical rules to reach a conclusion, demonstrating how basic AI systems operate.

---

## 💻 Technologies Used

* Python
* Conditional Logic
* Basic String Processing

---

## 📊 Example

Input: Secure@123  
Output: Strong


---

## ⚠️ Limitations

* The system relies on fixed rules
* It may not detect advanced password patterns
* It does not learn from new data
